# Hagoromo

Hagoromo (inspired by the legendary brand of [chalk](https://hagoromo.shop/)) is rusty version of Haskell's [diagrams](https://hackage.haskell.org/package/diagrams) library -- which lets you create vector graphics via a declarative DSL.

# Motivation

[Haskell](https://www.haskell.org/) is amazing. [Diagrams](https://hackage.haskell.org/package/diagrams) is amazing too. And yet, for real-world projects, [Rust](https://rust-lang.org/) is often a more pragmatic choice.

However, I was not able to find a comparable library in the Rust ecosystem, hence, hagoromo was born.
